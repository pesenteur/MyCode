## Enhanced Urban Region Embedding with Multi-scale Spatial Integration Network

This is a implementation of MSIN.

### Requirements

- Python 3.10.12
- PyTorch 2.3.1
- NumPy 1.26.4
- Scikit-learn 1.3.2
- Torch-Geometric 2.5.3
- SciPy 1.13.1

### Start
Run the command below to train the MSIN.
```
python training.py
```
